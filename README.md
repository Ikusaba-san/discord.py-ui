# discord.py UI

## WIP

An extension package for anything relating to user input and interactions.

## Installation

**Requires Python 3.6+**

### Development version
```
pip install git+https://github.com/Ikusaba-san/discord.py-ui.git
```
or
```
git clone git+https://github.com/Ikusaba-san/discord.py-ui.git
cd discord.py-ui
pip install -U .[voice]
```
